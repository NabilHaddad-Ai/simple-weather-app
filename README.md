A small vanilla JavaScript weather app that fetches data from the OpenWeatherMap API based on a city name.

I'm using this project to learn:
- How to call APIs from the browser
- How to work with async/await and promises
- How to structure a small front-end project
- How to use Git and GitHub

## Tech Stack

- HTML
- CSS
- JavaScript (ES6+)
- OpenWeatherMap API
- Bootstrap Icons

## Setup

1. Clone the repo.
2. Copy `config.example.js` to `config.js`.
3. Put your OpenWeatherMap API key in `config.js`.
4. Open `index.html` in your browser.
